Per aggiungere/togliere mappe: modifica l'array MAPS in sketch.js
(la lista di numeri) e usa file con lo stesso schema di nome,
oppure usa l'opzione "Personalizzata…" per caricarne una al volo.

Il displacement è visibile con Deform. X / Y > 0 (default 30).
Mappe quadrate (es. 1000x1000) consigliate.
